아두이노 기능

1. 비밀번호 입력 기능.
2. 비밀번호 맞게 입력했을 때 잠금장치 개폐.
3. 아두이노가 정상작동시 LED가 켜져있음.
4. MySQL이랑 무선연동.
  - 개폐상태전달
  - 비번 변경 가능

------------------------------------------------------

  기능 정리

  1. 사물함
    - 일정시간 뒤에 서버로 사물함 상태를 전송, 외관에 LED를 달아 그 상태를 표시
    - 안을 비추는 LED등을 설치
    - 사용자의 요구에 따라 원격으로 개폐
  2. 사용자 어플
    - 비번 관리 (변경 등)
    - 원격 제어 (개폐)
    - 열려 있는지 닫혀 있는지 확인 가능
    - 허가되지 않은 접근에 알림이 수신
    - 공지 수신
    - 자신의 사물함 정보를 확인
    * 적외선으로 물건이 있는지 확인(보류로 결정.)
  3. 관리자 어플
    - 등록된 다수의 사물함의 개폐 통제 가능
    - 사용자 어플에 공지 송신
    - 비밀번호 임시 변경
    - 사물함의 이상 발생 시 알람이 수신
  4. 총괄 서버
    - 사물함에서 신호가 오지 않을 경우 관리자에게 알림을 송신
    - 사용자에게서 비밀번호를 받아 확인 후 사물함을 개폐
    - 관리자의 요구 시 사물함을 개폐
    - 어플에서 비밀번호를 수정 시 반영
    - 사물함의 개폐 상태를 사용자에게 수신
차별점.
  - 키패드로 열고 닫기 가능.
  - 관리자가 다수 사물함을 한 번에 여닫을 수 있음.
  - 원격 개폐
  - 어플을 이용하여 사물함 상태 확인.
