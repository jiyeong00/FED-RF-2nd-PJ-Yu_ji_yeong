[ DC PJ 설치스크립트 ]
1. 리액트 앱 설치하기
- 설치위치: 03.DC_PJ\02.구현소스
- 설치이름: art-app
- 스크립트: npx create-react-app art-app

2. 팩키지 모듈 설치하기
- 설치위치: 03.art_PJ\02.구현소스\art-app
- 스크립트: 
    (1) npm i react-router-dom@latest
    (2) npm i jquery
    (3) npm i jquery-ui-dist
    (4) npm i jquery-ui-touch-punch
    (5) npm i swiper
    npm i @fortawesome/fontawesome-svg-core
    npm i @fortawesome/free-solid-svg-icons @fortawesome/free-regular-svg-icons @fortawesome/free-brands-svg-icons
    npm i @fortawesome/react-fontawesome
    npm i sass
    
3. 설치확인
- 파일명: package.json
- 항목명: "dependencies"

4. 개발서버구동
- 서버구동위치: 03.art_PJ\02.구현소스\art-app
- 스크립트:
    (1) 시작: npm start
    (2) 멈춤: ctrl+c 후 질문에 답: y
